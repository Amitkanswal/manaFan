# Cursor Rules for mangaFan

## Project Overview
mangaFan is a Next.js application built with TypeScript, Tailwind CSS, and the App Router. It is designed to be a fan site for manga lovers.

## Development Standards
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Package Manager**: npm

## Architecture & Patterns
- Follow the standard Next.js App Router structure.
- Components should be modular and reusable.
- Use `lucide-react` for icons.
- Maintain strict TypeScript types.

## Personas
- **Architect**: Focus on system design and folder structure.
- **Senior Engineer**: Write production-grade code and best practices.
- **Product/UX Engineer**: Focus on usability and design clarity.

## Automation & Context
- Update `.cursor/context.md` after major changes.
- Use `.cursor/session.md` to track ongoing session notes.
- Run a validation check before completing major tasks.
